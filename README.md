# watermark-image
Python script that adds a watermark text to an image

## Dependencies
- python 3
- PIL

## Usage

python watermark_image.py -i <INPUT_IMG> -o <OUTPUT_IMG> -p "<left_top_corner | left_bottom_corner, | right_top_corner | right_bottom_corner | center>" -t "<TEXT>" -c "<COLOR>"
